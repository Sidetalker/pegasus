# Trait generation alternatives

The options below can complement or replace OpenArt for trait generation. Review pricing,
licensing, and API terms before adoption.

## Candidates
- **Stability AI / Stable Diffusion API**: Hosted API for text-to-image and image-to-image workflows.
- **Leonardo AI**: Asset generation with model fine-tuning and API access for content pipelines.
- **Scenario**: Model training and generation focused on game assets; exposes API endpoints.
- **Krea**: Real-time generation with APIs for workflows and style transfer.
- **FAL (fal.ai)**: Hosted inference for SDXL and other models with SDKs and APIs.
- **Replicate**: Model hosting marketplace with API access for SDXL, ControlNet, etc.

## Evaluation checklist
- API availability for batch generation and seed control.
- Licensing terms for commercial NFT usage.
- Support for transparent PNG output and alpha channels.
- Cost per image and rate limits for large collections.
- Ability to fine-tune on Pegasus traits or style guides.
